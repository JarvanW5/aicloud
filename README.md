# AICloud
AI Cloud一键式生成所有大模型结果，支持近百种大模型的调用，还有社区交流和讨论功能

## 使用技术

- Spring Boot
- Spring Cloud Alibaba
- MySQL
- Mybatis-Plus
- Redis
- Kafka
- Spring Cloud Alibaba Nacos
- Spring Cloud Alibaba Sentinel
- Spring Security
- XXL-Job
- JWT
- MinIO
- 并发编程
- 分布式锁
- Sharing-JDBC（分库分表）
- Canal
- Hutool
- Spring Validation
- Swagger
